This game was developed for the **Bread Jam 2** game jam, which you can check out on [Itch.io](https://itch.io/jam/bread-jam-2).

Mini roguelike bread survivor made for Bread Jam 2  
*(Made in under 48 hours)*

---

## Controls

- **Arrow Keys** – Move (stay still to attack)
- **Z / C / N** – 🅾️ Button
- **X / V / M** – ❎ Button
